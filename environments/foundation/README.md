# Foundation Example

Sandbox environment for Lesson 1 — Foundations. Includes a starter `CLAUDE.md` so learners can see how project memory shapes an agent's behavior from the first turn.

## Prerequisites


## Running with npm

```bash
npm install
npm start
```

## Running with Devcontainer

1. Open this folder in VS Code
2. Click "Reopen in Container" when prompted
3. Run: `npm start`

## CLAUDE.md

The `CLAUDE.md` at the root of this folder is loaded automatically by Claude Code on every session. Edit it, then start a new Claude Code session here and notice how the agent's defaults change.
